# 🎓 Watsonx Campus Assistant

## 🚀 Overview / Project Overview
The Watsonx Campus Assistant is an intelligent, Generative AI-powered digital helpdesk specifically designed for the United University student portal. Navigating university resources can sometimes be time-consuming for students. This project solves that by acting as a 24/7 virtual assistant that understands natural language queries. It provides students with immediate, accurate responses regarding their campus life, bridging the gap between students and university administration through seamless AI integration.

## 🛠️ Tools & Technologies Used
* **AI & NLP Engine:** IBM watsonx (Generative AI)
* **Frontend Development:** HTML5, CSS3, JavaScript
* **Integration:** IBM Watson Web Chat UI
* **Version Control:** Git & GitHub

## 💡 Key Insights Generated
* **Streamlined Academic Access:** Centralized access to BCA-IBM program materials, reducing the time students spend searching for syllabi and course details.
* **Placement Readiness:** Automated the delivery of placement statistics and eligibility criteria, ensuring students are always updated on career opportunities.
* **Enhanced User Engagement:** Demonstrated that an interactive chat UI significantly improves student interaction with the university's digital portal compared to traditional static web pages.
* **Query Resolution Automation:** Successfully automated answers for the most frequently asked administrative and campus facility questions.
